### Pet Projects
1. Link Tracker - https://github.com/mkhalez/link-tracker  
Агрегатор уведомлений, помогающий в одном месте управлять уведомлениями с множества веб-ресурсов, таких как GitHub и Stack Overflow. Является многомодульным проектом, состоящим из 3 микросервисов: Telegram-бота, общающегося с пользователями, scrapper-сервиса, работающего с внешним API, и AI-agent, структурирующего сообщения для пользователей. Также используется СУБД PostgreSQL, брокер сообщений Kafka, а общение между ботом и scrapper происходит на основе gRPC. В проекте активно используются интеграционные и компонентные тесты. Сам проект сейчас находится в активной стадии разработки.

2. Labyrinths - https://github.com/mkhalez/labyrinths  
Консольная утилита на основе библиотеки Picocli, позволяющая строить лабиринты на основе остовного дерева и поиска в ширину, а также предназначенная для нахождения кратчайшего пути с использованием алгоритмов A* и Дейкстры.

3. Fractal-flame - https://github.com/mkhalez/fractal-flame  
Консольная утилита, предназначенная для генерации фракталов на основе идеи фрактального пламени. Генерация работает в двух режимах: однопоточном и многопоточном. В многопоточном режиме используются атомарные типы, такие как `LongAdder`, позволяющие более эффективно считать суммы, в отличие от базовых атомарных типов, таких как `AtomicLong`.

4. Log-analyzer - https://github.com/mkhalez/log-analyzer  
Консольная утилита для анализа логов NGINX, предназначенная для подсчета статистики из логов. Активно используются регулярные выражения, поддерживается парсинг как локальных файлов, так и удалённых ресурсов.
